Copy as Markdown In this article Copying and pasting cells in a table Dragging a cell&#x27;s contents to set neighboring table cells Clearing the content of multiple table cells Undoing bulk changes in a table Copying and pasting cells in a table You can copy the contents of one cell and paste the contents into multiple other cells in the same field. Navigate to a project view that is using the table layout. Select the cell that you would like to copy. To copy the contents of the selected cell, press Command + C (Mac) or Ctrl + C (Windows.) Select the destination cells for the copied content. Hold Command (Mac) or Ctrl (Windows/Linux) and select each cell. Select a cell then press Shift + ↑ or Shift + ↓ to select additional cell above or below the selected item. Select a cell then press Shift and select another item to select all items between the two items. Holding your mouse button down, move the cursor over the cells you want to select. To paste the previously copied content, press Command + V (Mac) or Ctrl + V (Windows.) Dragging a cell's contents to set neighboring table cells You can quickly copy one cell's content over other cells in the same field by dragging up or down. Navigate to a project view that is using the table layout. Select the cell that you would like to copy. If it's possible to copy the cell's content, a blue handle will be shown on the lower right of the selected cell. Drag the handle up or down over the cells where you would like to copy the contents. When you finish dragging, the contents of the originally selected cell will be copied over the cells you selected while dragging. Clearing the content of multiple table cells You can quickly clear the same field for multiple items. Navigate to a project view using the table layout. Select the cells you would like to clear. Hold Command (Mac) or Ctrl (Windows/Linux) and select each cell. Select a cell then press Shift + ↑ or Shift + ↓ to select additional cell above or below the selected item. Select a cell then press Shift and select another item to select all items between the two items. Holding your mouse button down, move the cursor over the cells you want to select. Press Delete or Backspace on your keyboard. Undoing bulk changes in a table When you make a bulk change in the table layout, GitHub will display the option to undo that change.
