# Stream Deck MK.2 Technical Specifications

| Property | Value |
|----------|-------|
| **Model** | Stream Deck MK.2 |
| **DeviceModel** | `20GBA9901` / `20GBA9902` |
| **Grid** | 5 columns x 3 rows |
| **Total Keys** | 15 |
| **Key Size** | 72x72px (144x144 @2x retina) |
| **LCD Type** | Customizable LCD keys |
| **Connection** | USB-C |
| **Dimensions** | 118 x 84 x 25 mm |
| **Weight** | ~145g |
| **OS Support** | Windows 11+, macOS 13+ |
| **Faceplate** | Detachable, magnetic |

## Image Dimensions Note

The official spec says 72x72px per key, but the Stream Deck app uses **144x144px** (@2x retina). Always use 144x144 for button icons in profiles.
