# Contributing to OpenDeck Factory

Contributions accepted under the Functional Source License, Version 1.1
(FSL-1.1-ALv2) via the Developer Certificate of Origin (DCO).

By contributing you agree that your contribution is licensed under FSL-1.1-ALv2
and will auto-convert to Apache License, Version 2.0 two years after the
version containing your contribution is first made available.

## Developer Certificate of Origin

By contributing, you certify your contribution under the terms of the
[DCO 1.1](https://developercertificate.org/).

## Signing off commits

Every commit requires a `Signed-off-by` trailer with your real name:

    Signed-off-by: Jane Doe <jane@example.com>

Use `git commit -s` to add it automatically. Anonymous sign-offs not accepted.

## Pull requests

1. Fork and branch from `master`.
2. Make changes with signed-off commits.
3. Run `node src/index.js validate` against affected profiles.
4. Open a PR describing the change.

## Scope

This repository is the open-source engine only. The commercial asset catalog
is maintained separately under proprietary license and is closed to external
contribution.

Welcome contributions:
- Engine bug fixes and APIs
- New device support
- Key code/platform expansions
- Template improvements
- Documentation
